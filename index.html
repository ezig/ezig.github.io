Budget Butler

  • App Store

  • GitHub

A polite app to streamline your budget management experience. Available free on the app store. Budget Butler is designed to simplify keeping track of receipts and spending by making it easy to add new receipts as you get them and to view past spending, either in table or chart form.

Adding a picture of a new receipt automatically fills in the total amount using optical character recognition powered by Tesseract. Receipts pictures and information are then saved using Core Data. Budget Butler will even fill in payee names and expense categories based on previous saved purchases. In addition to the table-style log, the summary view generates a pie chart using Core Plot to break down spending by category.

Strategy Game Movement

  • Live Demo

  • GitHub


An experiment with path-finding similar modeled on the way that movement works in turn-based strategy games like Fire Emblem and Advance Wars. Determines valid movement range based on a character's movement range and finds paths to tiles using a modified depth-limited breadth-first search that finds the shortest path to each tile. Built using Phaser.js to draw grid and move sprites.

To try out the demo, click on and move the pink or the green friendly sprites to a valid (blue-highlighted) tile in movement range. Once you've moved both friendly sprites, the red enemy will make a move. The red-highlighteds squares indicate spaces within the sprite's attack range but outside of its move range
